# Module Playbook - ClubGG Agent Tools Landing Page

## Landing Page Module
- **Standards**:
  - Use semantic HTML (header, main, section, footer).
  - Accessibility: `alt` tags for images, clear contrast, focus states.
  - Performance: Optimize any assets (images, icons).
  - Responsive: Use Tailwind's `sm`, `md`, `lg`, `xl` breakpoints.
  - Buttons: Use a consistent primary/secondary style.
  - Typography: Inter font, consistent scale.
