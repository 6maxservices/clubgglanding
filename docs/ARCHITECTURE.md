# Architecture - ClubGG Agent Tools Landing Page

## Tech Stack
- **Framework**: React 18 (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: TypeScript

## Structure
- `src/components`: Reusable UI components (Hero, Features, etc.)
- `src/App.tsx`: Main page assembly.
- `src/index.css`: Global styles and Tailwind directives.

## Design Principles
- **Dark Mode**: Professional, SaaS-style aesthetic (#0f172a).
- **Mobile First**: Fully responsive design.
- **Performance**: Minimal dependencies, fast loading.
